# 🌐 NovaTech – Premium Responsive Landing Page

> A modern, elegant, and fully responsive landing page built with **HTML5, CSS3, and JavaScript**.

NovaTech is a startup-inspired landing page designed to provide a premium user experience across desktops, tablets, and mobile devices. The project focuses on responsive web design, clean UI, smooth animations, and modern layouts using CSS Media Queries.

---

## ✨ Features

- 🎨 Modern Glassmorphism Navigation Bar
- 📱 Fully Responsive Design
- 🚀 Premium Hero Section
- 💎 Interactive Feature Cards
- 📊 Statistics Section
- 👨‍💻 About Us Section
- 📩 Contact Form
- 🌙 Professional Footer
- ✨ Smooth Scrolling
- 🎯 Hover Effects & Animations
- ⚡ Optimized Performance
- 📐 Mobile, Tablet & Desktop Support
